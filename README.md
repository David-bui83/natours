# NATOURS
---
Project for tours

## Useage
> To install packages run:
> - npm install

> Start application:
> - npm run start

### Dependencies

>Packages required to run program
> - express
> - morgan
> - mongoose
> - slugify
> - validator
> - bcryptjs
> - jsonwebtoken
> - nodemailer
> - express-rate-limit
> - helmet
